📖 关于我们
 
Airluxi 是面向模拟飞行爱好者的社区型虚拟航空公司。
我们汇聚各地飞行玩家，致力于打造真实、友好、注重团队氛围的模拟飞行社区。
 
我们提供以下服务与内容：
 
- ✈️ 高还原度的虚拟航空运营体系
- 📅 完整航线排班与机队运营
- 🛫 飞行员招募、入门指导与飞行培训
- 📝 飞行记录收录与社区活动策划
- 🎨 自制涂装制作、机队专属涂装支持
 
无论你是刚入门的飞行新人，还是经验丰富的模拟飞行玩家，
你都可以加入我们，一同成长、享受飞行。
 
🤝 加入我们
 
如果你希望成为 Airluxi 航空的一员：
 
1. 进入官方网站，完成飞行员注册流程
2. 阅读航空运行规范与飞行员手册
3. 在 GitHub 社区讨论区提问、交流、获取最新通知
 
我们长期招募：飞行飞行员、网站开发者、内容创作者、社区运营成员。
